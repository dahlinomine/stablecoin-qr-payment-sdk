## Overview

**stablecoin-qr-payment-sdk** is an open-source sdk for stablecoin infrastructure and digital payments.

TypeScript SDK for generating and processing interoperable QR codes for stablecoin payments.

Topics: `qr-codes` `payments` `mobile`

## Features

- 🚀 Production-ready sdk targeting stablecoins use cases
- 🔧 Easy to integrate into existing projects
- 📦 Zero-configuration defaults with full customization support
- 🧪 Comprehensive test coverage
- 📖 Well-documented API with examples
